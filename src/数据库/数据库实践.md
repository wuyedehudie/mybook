- 0.注释
- 查询所有活跃用户
```
 SELECT *
FROM users
WHERE status = 'active';
```
- 1.WHERE
```
WHERE category = 'A'
  AND status = 'active'
```
- 2.子查询缩进
``` 
SELECT column0, column2
FROM (
    SELECT column0, column2
    FROM my_table
    WHERE condition
) AS subquery
WHERE subquery.column0 > 100;
```
